✨ Key Features

  • Centralized source of truth — Model sheets, reference packs, and prompt modules in one versioned repository
  • Prompt-engineering ready — Assets organized by scenes, characters, and shot types for easy pipeline integration
  • Version-controlled visual canon — Character details refined via commits, ensuring tight visual continuity
  • Collaboration-friendly — Designed to be forked, extended, and improved by the community
  • Platform-agnostic — No more oversized zips or Discord upload limits; everything is Git-friendly
